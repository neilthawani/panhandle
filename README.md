# panhandle

Compile Handlebars Templates in a Github Pages Static Site

### Requirements

- [Node.js](nodejs.org)

### Installation Steps

1. Clone this repository.
2. Run `npm install`
3. Install the module globally using `npm install -g`
4. Run `panhandle new <project-name>`
